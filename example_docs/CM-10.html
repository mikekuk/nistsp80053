CM-10: SOFTWARE USAGE RESTRICTIONS

Control Description

The organization:

  1. CM-10a. Uses software and associated documentation in accordance with contract agreements and copyright laws;
  2. CM-10b. Tracks the use of software and associated documentation protected by quantity licenses to control copying and distribution; and
  3. CM-10c. Controls and documents the use of peer-to-peer file sharing technology to ensure that this capability is not used for the unauthorized distribution, display, performance, or reproduction of copyrighted work.

Supplemental Guidance

Software license tracking can be accomplished by manual methods (e.g., simple spreadsheets) or automated methods (e.g., specialized tracking applications) depending on organizational needs.
