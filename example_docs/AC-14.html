AC-14: PERMITTED ACTIONS WITHOUT IDENTIFICATION OR AUTHENTICATION

Control Description

The organization:

  1. AC-14a. Identifies [Assignment: organization-defined user actions] that can be performed on the information system without identification or authentication consistent with organizational missions/business functions; and
  2. AC-14b. Documents and provides supporting rationale in the security plan for the information system, user actions not requiring identification or authentication.

Baselines

LOW, MODERATE, HIGH
