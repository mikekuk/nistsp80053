AC-11: SESSION LOCK

Control Description

The information system:

  1. AC-11a. Prevents further access to the system by initiating a session lock after [Assignment: organization-defined time period] of inactivity or upon receiving a request from a user; and
  2. AC-11b. Retains the session lock until the user reestablishes access using established identification and authentication procedures.

Control Enhancements

  • AC-11 (1) - PATTERN-HIDING DISPLAYS

    The information system conceals, via the session lock, information previously visible on the display with a publicly viewable image.

Baselines

MODERATE, HIGH
