PE-17: ALTERNATE WORK SITE

Control Description

The organization:

  1. PE-17a. Employs [Assignment: organization-defined security controls] at alternate work sites;
  2. PE-17b. Assesses as feasible, the effectiveness of security controls at alternate work sites; and
  3. PE-17c. Provides a means for employees to communicate with information security personnel in case of security incidents or problems.

Baselines

MODERATE, HIGH
