CA-9: INTERNAL SYSTEM CONNECTIONS

Control Description

The organization:

  1. CA-9a. Authorizes internal connections of [Assignment: organization-defined information system components or classes of components] to the information system; and
  2. CA-9b. Documents, for each internal connection, the interface characteristics, security requirements, and the nature of the information communicated.

Control Enhancements

  • CA-9 (1) - SECURITY COMPLIANCE CHECKS

    The information system performs security compliance checks on constituent system components prior to the establishment of the internal connection.

Baselines

LOW, MODERATE, HIGH
