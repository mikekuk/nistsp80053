PE-10: EMERGENCY SHUTOFF

Control Description

The organization:

  1. PE-10a. Provides the capability of shutting off power to the information system or individual system components in emergency situations;
  2. PE-10b. Places emergency shutoff switches or devices in [Assignment: organization-defined location by information system or system component] to facilitate safe and easy access for personnel; and
  3. PE-10c. Protects emergency power shutoff capability from unauthorized activation.

Supplemental Guidance

This control applies primarily to facilities containing concentrations of information system resources including, for example, data centers, server rooms, and mainframe computer rooms.
