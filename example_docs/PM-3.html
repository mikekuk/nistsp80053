PM-3: INFORMATION SECURITY RESOURCES

Control Description

The organization:

  1. PM-3a. Ensures that all capital planning and investment requests include the resources needed to implement the information security program and documents all exceptions to this requirement;
  2. PM-3b. Employs a business case/Exhibit 300/Exhibit 53 to record the resources required; and
  3. PM-3c. Ensures that information security resources are available for expenditure as planned.
