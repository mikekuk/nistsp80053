SA-3: SYSTEM DEVELOPMENT LIFE CYCLE

Control Description

The organization:

  1. SA-3a. Manages the information system using [Assignment: organization-defined system development life cycle] that incorporates information security considerations;
  2. SA-3b. Defines and documents information security roles and responsibilities throughout the system development life cycle;
  3. SA-3c. Identifies individuals having information security roles and responsibilities; and
  4. SA-3d. Integrates the organizational information security risk management process into system development life cycle activities.

Baselines

LOW, MODERATE, HIGH
