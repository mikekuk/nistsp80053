PE-5: ACCESS CONTROL FOR OUTPUT DEVICES

Control Description

The organization controls physical access to information system output devices to prevent unauthorized individuals from obtaining the output.

Control Enhancements

  • PE-5 (1) - ACCESS TO OUTPUT BY AUTHORIZED INDIVIDUALS

    The organization:

    1. PE-5 (1)(a) Controls physical access to output from [Assignment: organization-defined output devices]; and
    2. PE-5 (1)(b) Ensures that only authorized individuals receive output from the device.

  • PE-5 (2) - ACCESS TO OUTPUT BY INDIVIDUAL IDENTITY

    The information system:

    1. PE-5 (2)(a) Controls physical access to output from [Assignment: organization-defined output devices]; and
    2. PE-5 (2)(b) Links individual identity to receipt of the output from the device.

  • PE-5 (3) - MARKING OUTPUT DEVICES

    The organization marks [Assignment: organization-defined information system output devices] indicating the appropriate security marking of the information permitted to be output from the device.

Baselines

MODERATE, HIGH
